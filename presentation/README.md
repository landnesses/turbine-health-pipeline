# Wind Turbine AI Decision Support — Web Presentation

A modular web-based presentation for the Scottish Power wind turbine AI decision support system.

## Project Structure

```
Scottish_Power_pre/
├── index.html          # Generated by build (main entry)
├── build.js            # Build script
├── css/
│   └── style.css       # All styles
├── js/
│   └── script.js       # Presentation logic
├── slides/
│   ├── manifest.json   # Slide order, titles, notes
│   ├── 01-title.html   # Slide 1 content
│   ├── 02-problem.html # Slide 2 content
│   └── ...             # One file per slide
├── plots/              # RL charts (episode_comparison.png, reward_distribution.png)
│
│ Icons: Font Awesome 6 (CDN). Custom SVGs retained for Pipeline architecture diagram and Title hero.
└── README.md
```

## Quick Start

1. **Build** (required after editing slides):
   ```bash
   node build.js
   ```
   Outputs to `public/` (used by Vercel). For local preview, open `public/index.html` or serve the `public/` folder.

2. **View**: Open `index.html` in a browser, or run a local server:
   ```bash
   npx serve .
   # or
   python -m http.server 8000
   ```

## Editing Slides

- **Modify content**: Edit `slides/01-title.html`, `slides/02-problem.html`, etc.
- **Change order or metadata**: Edit `slides/manifest.json` (id, title, notes, file)
- **Add a slide**: Create `slides/12-new-slide.html`, add entry to `manifest.json`
- **Rebuild**: Run `node build.js` after changes

## Vercel Deployment

The project includes `package.json` with a `build` script. Vercel will automatically run `npm run build` on each deploy, regenerating `index.html` from the slides.

No additional configuration needed — connect your repo and deploy.

## Keyboard Shortcuts

- `↑` / `↓` or `PgUp` / `PgDn` — Navigate slides
- `P` — Toggle presenter mode
- `A` — Toggle auto-play
- `F` — Fullscreen
